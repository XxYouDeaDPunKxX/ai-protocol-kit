what this is

A practical map for choosing how the AI should behave before the work starts.

These are not magic prompts. They are working instructions you give to the AI before the task, so it knows what to check, what not to assume, when to stop, and what kind of output is acceptable.

how to use

Give the protocol before the work starts.

  1. 1Pick the closest path.
  2. 2Give the protocol to the AI.
  3. 3State the task and hard constraints.
  4. 4Let it guide questions, review, and output.

Use the smallest set that fits the task. Do not stack protocols casually.
